# Dic Multiple

在 macOS 原生词典中一次性查询多个单词且并排显示，方便比对词义、斟酌用词。

调用原生词典批量查词。使用时，选中一串**空格为界**的单词，稍等几秒——因为启动词典软件本身需要时间——即可打开多个查词窗口，每个单词查一次，所有词条都摆在桌面上。

出处：[《在 macOS 原生词典中批量查词（多窗口）》](https://utgd.net/article/20530)
